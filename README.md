## 🎯 主な実績

### 大手マッチングアプリ開発案件
- Laravel × React（Inertia）× Tailwind CSS × WebSocket（Reverb）によるリアルタイムチャット機能開発
- データベース設計の一部、バックエンド、フロントエンド、リアルタイム配信まで対応

### メディパ様（美容医療）サービス全般制作
- Googleのビジネスコンテストで優勝したチームのサービス
- WordPress、PHPバックエンド設計、サービス導線実装

### 串かつ さじろう様 リニューアル
- 6ページをCMS化まで5日で実装
- WordPress、オリジナルテーマ、フルスタック

### 五料産業株式会社様 リニューアル
- ヒアリング〜実装まで一貫対応
- Figma、デザイン、複数ページ実装、フルスタック

### HEART&BRAIN様 Executive Compass LP制作・既存サイト改修
- WordPress既存サイト内に新サービスLPを制作
- 上場企業の役員層が閲覧する前提で、信頼感と既存サイトとの統一感を重視
- AI生成の初期案から公式サイト掲載に耐えるLPへ再構成
- テストサイト構築、FTP作業、前制作会社からの引き継ぎ、メインサイト修正に対応

### LP制作 1ヶ月8本ペース・フルスタック実装
- ビジネスLPを1ヶ月8本ペースで制作・改修
- デザイン、フロントエンド、バックエンド、ログイン、新規登録、会員情報、データベース、決済導線まで担当
- Note Partnerを中心に、サービス運用に必要な会員機能・マイページ・支払い情報入力まで実装

### カタログ制作 業務効率化ツール
- PythonとAI画像認識を活用し、花のカタログ画像・掲載決定書・商品マスタの照合を効率化
- カタログ画像をテキスト化し、Excelとの差分・不一致候補を抽出
- NDAのため実データは非公開。生成画像と抽象化した説明で掲載

### 書籍横断検索アプリ
- 複数の図書館・書店を一括検索できるWebアプリ
- マイページ導線を追加し、ログイン後の利用性も改善
- Python、API、スクレイピング、Streamlit

### VEIN ENERGY様 コーポレート
- ロゴ制作から実装まで2日で納品
- STUDIO、ロゴ制作、STUDIO実装

## 🚀 セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/Oota-0505/Portfolio20260101.git
cd Portfolio20260101
```

2. ローカルサーバーで起動
```bash
# Python 3の場合
python -m http.server 8000

# Node.jsの場合
npx http-server
```

3. ブラウザでアクセス
```
http://localhost:8000
```
