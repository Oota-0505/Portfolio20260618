EthioPick

エチオピア産コーヒー豆だけに特化した、レビュー・抽出メモ共有アプリ。豆を買ったホームバリスタが「この豆をどう淹れたら一番おいしかったか」を共有する個人プロダクト。

Laravel Inertia.js React Reverb MariaDB 個人開発

概要

エチオピア産コーヒー豆だけに特化した、レビュー・抽出メモ共有アプリです。 一般的な「コーヒー店レビュー」ではなく、豆を買ったエンドユーザー（ホームバリスタ）が「このエチオピア豆をどう淹れたら一番おいしかったか」を共有することにフォーカスしています。

YouTube学習で作ったコーヒー店レビューアプリを土台に、エチオピア豆レビュー中心のプロダクトへ育てている個人開発です。日々「React学習＋アプリ改善」を継続し、最終的な公開を目指しています。（※現状の画面は店舗レビューの土台で、豆特化へ移行中）

コンセプト・背景

  • 対象は、エチオピア産コーヒーが好きな人・ホームバリスタ
  • 豆を買ったあと「よりおいしく淹れる方法」を探している人に向ける
  • ドリッパー、挽き目、湯温、レシピ、味のメモなどの抽出ノウハウを共有する
  • 「お店の評価」はせず、豆と淹れ方に特化する（ニッチに絞る）

主な機能（構想・MVP）

  • エチオピア豆のレビュー投稿（豆名・焙煎所・購入先・精製方法・焙煎度）
  • 総合評価と味のメモ
  • 抽出方法の記録（器具・レシピ・挽き目・湯温など）
  • 口コミの良いエチオピア豆を見つけやすくする
  • 各コーヒー豆ごとに、掲示板的なやり取り（コメント・淹れ方共有）（予定）
  • ユーザー新規登録（会員機能）（予定）

技術スタック

Backend

PHP 8.3 / Laravel

Frontend

Inertia.js + React / Vite

Realtime

Laravel Reverb + Echo（WebSocket）

DB / UI

MariaDB / Chakra UI・Tailwind

このスタック（Laravel + Inertia + React + Reverb）は、実際の案件（大手マッチングアプリ）とほぼ同じです。仕事の技術と地続きの個人開発になっています。

こだわり・工夫

  • 「店レビュー」から「豆レビュー＋淹れ方共有」へ、プロダクトの軸を明確に定義し直した
  • ニッチ（エチオピア豆）に絞ることで、何のアプリか一目で分かる差別化
  • Chakra UIでフォーム・モーダル・ドロワーを高速に実装し、テーマでブランド色を共有
  • リアルタイム基盤（Reverb/Echo）を学習・導入
  • PRODUCT_VISION・DB設計・競合調査・収益化まで、ドキュメントで設計しながら開発

今後（構想・予定）

  • 店舗レビューの土台 → エチオピア豆特化へ、UI／データ構造を移行
  • ユーザー新規登録（会員機能）
  • 各コーヒー豆ごとの掲示板的なやり取り（コメント・淹れ方共有）
  • MVP公開（エチオピア豆の投稿＋評価＋味メモ）
  • 抽出レシピの記録・検索性の向上、交流機能の拡充
